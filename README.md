# Crypto Market Analysis Project

## Project Overview

This project focuses on analyzing cryptocurrency and financial market data using Python. The main objective was to explore market trends and visualize important insights using data analysis and a dashboard.

## Tools and Technologies Used

* Python
* Pandas for data analysis
* SQLite for database management
* Streamlit for building the interactive dashboard
* Google Colab for initial data processing and analysis

## What I Did in This Project

In this project, I collected and worked with datasets related to cryptocurrency, Bitcoin prices, oil prices, and stock market data. First, I performed data analysis using Python in Google Colab. I cleaned and processed the datasets and created tables using SQLite for structured data storage.

Then I performed various data queries and analysis to understand market trends and relationships between different financial indicators.

After the analysis, I built an interactive dashboard using Streamlit in Visual Studio Code. The dashboard displays important insights such as cryptocurrency market data and graphical trends of Bitcoin, oil, and stock prices. This helps in visually understanding how these markets change over time.

## Outcome

The final result of this project is a working Streamlit dashboard that presents financial data in an easy-to-understand graphical format, allowing users to explore market trends interactively.
